쇼츠 일일 운영 대시보드

매일 채널별 이슈, 후보 주제, 대본, 소스, 편집 구성을 한 페이지에서 관리하는 GitHub Pages용 템플릿

업데이트 날짜:
전체 대시보드 연예/방송/화제 글로벌 트렌드 지식형/랭킹형 동물/상품형

이 사이트로 할 수 있는 일

  • 채널별로 매일 주제를 분리해서 확인
  • 후보 주제, 최종 주제, 대본, 자막, 화면 구성을 한 번에 정리
  • GitHub Pages로 웹사이트처럼 열람
  • 나중에 자동화 붙일 수 있는 구조로 확장
처음에는 직접 내용을 수정해도 충분합니다. 익숙해진 뒤 Google Sheets, Notion, API 자동화를 붙이면 됩니다.

오늘 할 일

1단계
각 채널 페이지를 열어 오늘의 주제 후보 확인
2단계
최종 주제 1개씩 확정
3단계
대본과 자막을 복사해 편집 시작
4단계
업로드 후 성과는 별도 시트로 기록
